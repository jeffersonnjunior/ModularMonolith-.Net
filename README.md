# ModularMonolith-.Net

# Documentação do Projeto

## Regra de Negócio
Projeto para gerenciamento de fábrica de carros, com funcionalidades como controle de peças em estoque, gestão da linha de produção e acompanhamento de vendas de veículos.

## Tecnologias
- C#, ASP.NET Core, PostgreSQL, Redis, Entity Framework, xUnit e Docker

## Padrões de Design
- DDD, CQRS, SOLID, Decorator, Factory e Notification
